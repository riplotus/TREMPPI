The above table contains possible conditions for execution of additional tools, in case they have the select and/or compare functionality. Effect of chosen selections can be viewed in the tables above. The WHERE field describes the condition that must be met for a row to be selected. The condition may consist of atoms built using any of {<, <=, >, >=, =, !=}, logical connectives {AND, OR, NOT} and braces.
